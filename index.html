📥 Installation Requirements

Before Installing:

  1. Install Tampermonkey: If you haven't already, add the Tampermonkey extension to your browser:
    • Chrome Web Store
    • Firefox Add-ons
    • Edge Add-ons
  2. Verify Installation: Look for the Tampermonkey icon in your browser toolbar (usually top-right corner).
  3. Enable Scripts: Make sure Tampermonkey is enabled and set to "On" for the sites you want to use.
⚠️ Important: This script is designed specifically for Amazon's Harmony Console (console.harmony.a2z.com). Ensure you have proper access permissions before installation.

🎮 How to Use

  1. Navigate to Harmony Console: Go to console.harmony.a2z.com after installing the script.
  2. Access the Master Button: Look for the 📊 button in the bottom-right corner of your screen.
  3. Open Tool Folder Menu: Click the master button to see all available tools.
  4. Select Your Tool: Choose from Duranaut, TaskTally, Taskinator, Break Tracker, or IVE Guard.
  5. Configure P360 Notifications: Open Duranaut modal and adjust notification settings as needed.
  6. Start Tracking: Click "Get Tasks on Queue" to begin automatic time tracking.
  7. Monitor Progress: View real-time metrics and productivity indicators.

❓ Frequently Asked Questions

Does this work with P360 Timelog Monitor?

Yes! Version 15.00.01 includes full P360 integration. The script automatically syncs with P360 downtime tracking and displays a real-time timer widget.

Will my Taskinator history be preserved when I reset data?

Absolutely! The "Reset All Data" function specifically preserves Taskinator history while clearing all other productivity data.

How do I configure P360 notifications?

Open the Duranaut modal, scroll to the bottom, and you'll find P360 Notification Settings. You can enable/disable notifications and set the interval (1-60 minutes).

Can I use this on multiple browsers?

Yes, as long as Tampermonkey is installed. However, data is stored locally per browser, so you'll need to export/import if switching browsers.

What happens if I encounter errors?

Check the browser console (F12) for error messages. Most issues are resolved by refreshing the page or reinstalling the script.

How do I update to a newer version?

Simply click the install button again when a new version is released. Tampermonkey will prompt you to update, and your Taskinator history will be preserved.
